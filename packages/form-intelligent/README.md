# @jayoncode/form-intelligent (deprecated)

> **Use [`@jayoncode/form-intelligence`](https://www.npmjs.com/package/@jayoncode/form-intelligence) instead.**

This package is a **compatibility re-export** of Form Intelligence. It exists so existing installs and imports keep working.

```bash
npm uninstall @jayoncode/form-intelligent
npm install @jayoncode/form-intelligence
```

```ts
// before
import { createForm } from "@jayoncode/form-intelligent";

// after
import { createForm } from "@jayoncode/form-intelligence";
```
