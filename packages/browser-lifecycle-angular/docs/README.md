# Browser Lifecycle Angular

Official Angular adapter for [`@jayoncode/browser-lifecycle`](../browser-lifecycle/README.md).

Thin wrappers only — no browser observation logic lives here. See the package [README](../README.md) for install and usage.

## API surface

`provideBrowserLifecycle`, `injectBrowserLifecycle`, `createBrowserLifecycleHandle`
