# @jayoncode/form-intelligent-angular (deprecated)

> **Use [`@jayoncode/form-intelligence-angular`](https://www.npmjs.com/package/@jayoncode/form-intelligence-angular) instead.**

This package is a **compatibility re-export** of the Form Intelligence `angular` adapter.

```bash
npm uninstall @jayoncode/form-intelligent-angular
npm install @jayoncode/form-intelligence-angular
```

```ts
// before

// after
import { ... } from "@jayoncode/form-intelligence-angular";
```
