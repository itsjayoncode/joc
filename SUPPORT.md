# Support

## Browser Session Playground

The Browser Session Playground is the interactive engineering console for `@jayoncode/browser-lifecycle`.

- **Documentation:** http://127.0.0.1:4175 (local VitePress) or the deployed docs site
- **Playground:** http://127.0.0.1:4273 (local dev)
- **Release notes:** `apps/browser-session-playground/RELEASE_NOTES.md`
- **Known issues:** `apps/browser-session-playground/KNOWN_ISSUES.md`

## Getting help

1. Check the [FAQ](apps/docs/docs/packages/browser-lifecycle/faq/) in the documentation site
2. Review [troubleshooting](apps/docs/docs/packages/browser-lifecycle/troubleshooting/)
3. Search [existing issues](https://github.com/itsjayoncode/joc/issues)
4. Open a new issue with reproduction steps

## Bug reports

Include:

- Browser and version
- Playground route (for example `/configuration`)
- Steps to reproduce
- Expected vs actual behavior
- Console errors if any
