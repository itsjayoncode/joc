# Security Policy

## Supported Status

JOC is in an early repository foundation stage and no production-ready packages have been released yet.

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Instead, report the issue privately to the maintainers with:

- A clear description of the concern
- Reproduction details if applicable
- Impact assessment
- Suggested mitigation if known

The project will acknowledge reports, investigate responsibly, and coordinate disclosure when appropriate.

## Security Posture for This Phase

At this stage, the main focus is establishing sound repository and package boundaries so future packages can adopt consistent security practices as they are introduced.
