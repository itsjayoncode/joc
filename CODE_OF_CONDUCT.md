# Code of Conduct

## Our Commitment

JOC is committed to providing a respectful, welcoming, and harassment-free environment for everyone participating in the project.

## Expected Behavior

- Show empathy and professionalism.
- Assume good intent while staying clear and constructive.
- Give feedback about ideas and code, not about people.
- Respect different backgrounds, experience levels, and communication styles.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Personal attacks or hostile language.
- Publishing private information without consent.
- Disruptive behavior that undermines collaboration.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, and community spaces that represent the project.

## Reporting

If you experience or witness behavior that violates this code, report it privately to the project maintainers. Reports will be reviewed promptly and handled with care.

## Enforcement

Maintainers may remove, edit, or reject contributions and participation that do not align with this Code of Conduct.
