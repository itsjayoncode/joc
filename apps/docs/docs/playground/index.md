# Playground

Explore the **JOC Ecosystem** in the browser. Each playground is a standalone app bundled with the docs site — open a package below to try APIs, inspect state, and validate behavior without cloning the monorepo. Every package is thoroughly documented and backed by these interactive labs for a consistent developer experience.

<PlaygroundCatalog />

## Local development

Run a playground app directly from the monorepo when you are contributing or debugging:

```bash
pnpm browser-session-playground:dev
pnpm object-diff-playground:dev
pnpm form-intelligence-playground:dev
pnpm storage-playground:dev
```

See each package’s playground guide for routes, deployment notes, and module-specific walkthroughs.
