# Common Patterns

Production-ready patterns built on Browser Lifecycle.

| Pattern                                                  | Use case                             |
| -------------------------------------------------------- | ------------------------------------ |
| [Auto Save](/patterns/auto-save)                         | Persist drafts when the user pauses  |
| [Analytics](/patterns/analytics)                         | Emit presence-aware analytics events |
| [Presence Detection](/patterns/presence-detection)       | Track active vs idle users           |
| [Session Timeout](/patterns/session-timeout)             | Warn before session expiry           |
| [Background Sync](/patterns/background-sync)             | Sync when connectivity returns       |
| [Offline First](/patterns/offline-first)                 | Queue work while offline             |
| [Polling](/patterns/polling)                             | Pause polling when hidden            |
| [Page Tracking](/patterns/page-tracking)                 | Track page visibility sessions       |
| [Visibility Pause](/patterns/visibility-pause)           | Pause expensive work when hidden     |
| [Focus Resume](/patterns/focus-resume)                   | Resume on window focus               |
| [Shared WebSocket](/patterns/shared-websocket)           | One socket in the primary tab        |
| [Leader Election](/patterns/leader-election)             | Elect a single coordinator tab       |
| [Plugin Architecture](/patterns/plugin-architecture)     | Extend runtime behavior safely       |
| [State Synchronization](/patterns/state-synchronization) | Share state across tabs              |

Each pattern links to playground pages and framework examples where applicable.
