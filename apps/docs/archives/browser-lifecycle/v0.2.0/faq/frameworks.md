# Frameworks FAQ

## Are official adapter packages available?

Yes (experimental, workspace packages):

- `@jayoncode/browser-lifecycle-react`
- `@jayoncode/browser-lifecycle-vue`
- `@jayoncode/browser-lifecycle-svelte`
- `@jayoncode/browser-lifecycle-solid`
- `@jayoncode/browser-lifecycle-angular`

See [Framework adapters](/packages/browser-lifecycle/modules/adapters).

## Are framework examples available?

Yes. See [Framework Examples](/packages/browser-lifecycle/examples/) for Vanilla, React, Vue, Angular, Svelte, Next.js, Electron, and PWA demos in the monorepo.
